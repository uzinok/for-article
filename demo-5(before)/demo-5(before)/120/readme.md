`stroke-dasharray` закрашивает кольцо штрихом. В значении `"10 127"` первое число (`10`) задаёт длину штриха, второе (`127`) — длину промежутка; поскольку промежуток превышает длину окружности (≈125,66 единиц), на кольце отображается лишь один короткий штрих.
В следующем слайде длинная кривая закрашена таким же штрихом.
