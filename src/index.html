Чтобы создать текст, который будет следовать по окружности
необходимо нарисовать круг с помощью любого редактора SVG
Между кругом и краями изображения должно быть достаточно места для текста
Добавим text и textPath в SVG со своим текстом
Чтобы текст начинался сверу, используйне startOffset="50%"
А чтобы верхняя точка была центром текстового узла
используйте атрибут text-anchor="middle"
Подробнее в статье или видео
Ссылка в описании 
				
					<svg xmlns="http://www.w3.org/2000/svg"
					    width="240" height="240">
					  
					  <path fill="#ff6347"
					  d="M120 220a100 100 0 1 1 0-200 100 100 0 0 1 0 200z">
					  </path>
